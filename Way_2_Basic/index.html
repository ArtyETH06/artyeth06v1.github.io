Description: Here is some data represented in base 2. What is this data represented as ASCII text?

Type: Warmup

Level: Easy ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Ce challenge nosu demande de convertir des données en base 2 (binaire), en ASCII,voici la chaine de carcatère encodée en base 2: 01100110 01101100 01100001 01100111 01111011 00111001 00110000 01100011 00110110 01100101 01100010 01100101 00111001 00110100 00110001 00110101 00110110 00110001 01100011 01100110 01100001 01100100 01100110 01100001 01100101 00110001 00110111 00110000 01100001 00111000 01100110 00110000 01100101 01100001 00110010 00110101 00110010 01111101

Pour décoder cette chaine de caractère nous pouvons nous rendre sur: https://gchq.github.io/CyberChef/#recipe=From_Binary(‘Space’,8) Puis nous mettons le code en binaire et il nous renvoie le flag !
